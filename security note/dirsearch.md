Dirsearch 是一个用于探测[Web服务器](https://so.csdn.net/so/search?q=Web%E6%9C%8D%E5%8A%A1%E5%99%A8&spm=1001.2101.3001.7020)上的隐藏目录和文件的工具。它通过发送HTTP请求来尝试访问可能存在的路径，从而找到不列在网站目录页面上的隐藏资源。

### Dirsearch 的基本用法

启动程序前需确认 Python 环境配置正常以及依赖库均已加载完毕。以下是几个常用参数及其作用：

| 参数                            | 描述                                                                                            |
| ----------------------------- | --------------------------------------------------------------------------------------------- |
| `-h` 或 `--help`               | 显示帮助菜单                                                                                        |
| `-u URL`                      | 设置待测试的目标站点链接                                                                                  |
| `-L FILE`                     | 提供包含多个网址记录的文本清单作为输入源                                                                          |
| `-w WORDLIST`                 | 自定义探测所依据的关键字集合，默认位置位于 `/usr/share/wordlists/dirbuster/directory-list-lowercase.txt` 或随附于发行版之中 |
| `-r DEPTH`                    | 启动递归模式探索子层结构直至指定层数为止                                                                          |
| `-s SECONDS`                  | 调整两次连续请求间暂停间隔秒数以规避频率过高触发防护机制的风险                                                               |
| `-t THREADS`                  | 并发处理的最大线程数量提升效率的同时也要兼顾服务器负载承受能力                                                               |
| `-F`                          | 开启自动跳转跟踪确保即使存在重定位也不会遗漏实际响应内容                                                                  |
| `-c COOKIE_STRING`            | 添加必要的身份验证标记以便顺利进入受保护区域                                                                        |
| `-H HEADER_NAME:HEADER_VALUE` | 手动注入额外 HTTP 头部字段满足特殊场景需求比如绕过 WAF 检测规则等等…                                                      |
| `--timeout=TIMEOUT`           | 设定单次连接最大等待时限避免长时间无回应造成阻塞现象发生                                                                  |
| `--ip=IP_ADDRESS`             | 强制指派特定出口 IP 地址实现地理分布伪装效果减少被封禁概率                                                               |
| `-e EXTENSIONS`               | 过滤只关注某几种扩展名结尾的对象加快进度节省时间成本                                                                    |
| `--random-agents`             | 动态切换 User-Agent 隐藏真实客户端特征增加隐蔽程度                                                               |
